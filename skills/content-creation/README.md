# content-creation

Full-stack content creation pipeline skill for AI coding agents. Turns ideas into published blog posts, Remotion videos, and social media content.

```
npx skills add broomva/content-creation
```

## Pipeline

```
RESEARCH → NARRATIVE → VISUAL ASSETS → VIDEO → SOCIAL → DEPLOY
```

1. **Research** — gather evidence, screenshots, metrics
2. **Narrative** — structure with proven storytelling frameworks (ABT, PSI, McKinsey Pyramid)
3. **Visual Assets** — screenshots, GIFs, annotated images
4. **Video** — Remotion compositions from JSON specs
5. **Social** — X threads, Instagram carousels, LinkedIn posts
6. **Deploy** — publish to broomva.tech and distribute
