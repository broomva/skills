## [0.1.0] — 2026-06-09

Initial release.

### Added

- Algorave-grade livecoded music workflow — TidalCycles patterns (Haskell DSL →
  SuperDirt over OSC) + Hydra-synth visuals. Wraps the boot ritual
  (SuperCollider → SuperDirt → Tidal → Hydra), a vibe-descriptor pattern
  generator (industrial, ambient, DnB, footwork, techno, algorave-glitch), and a
  tunnel-visuals reference page. Mac-first; ports to Linux with package swaps.
