# colombia-conflict

A tested, installable **knowledge skill** distilled from the Colombian Truth
Commission (CEV) final report **_Hay Futuro Si Hay Verdad_** (2022) — 11 tomos /
24 books / ~8,900 pp / ~3.67M words.

It carries the report's substantive findings and gives agents tools to act on the
**problematics** the report describes — above all **no repetición**
(non-repetition).

## Install

```bash
npx skills add broomva/skills --skill colombia-conflict
```

## What's inside

| Path | What |
|---|---|
| `SKILL.md` | The contract + usage + when-to-reach-for-it. |
| `references/synthesis.md` | Master consolidated synthesis of the whole report. |
| `references/digests/*.md` | 12 sourced per-volume digests (the knowledge substrate). |
| `references/lexicon.md` | The report's coined-concepts glossary. |
| `references/knowledge-index.md` | Tier-1 routing catalog (generated by `cc.py index`). |
| `data/*.json` | Structured datasets: statistics, actors, 67 recommendations, concepts. |
| `scripts/cc.py` | The kg/LLM-wiki engine: `load`, `rec`, `stat`, `actor`, `concept`, `align`, `index`. |
| `tests/test_cc.py` | Unit + data-integrity tests (stdlib pytest, no network). |

## Quick start

```bash
python3 scripts/cc.py load "falsos positivos responsabilidad"   # two-tier retrieval
python3 scripts/cc.py rec --theme drug-policy                    # recommendations
python3 scripts/cc.py stat homicides                             # sourced statistic
python3 scripts/cc.py align "an agrarian reform with land restitution"  # non-repetition check
```

## Architecture

kg / LLM-wiki (LLM-as-index): substrate-canonical knowledge pages + a dense
catalog projection for tier-1 routing + body-grep tier-2 + **the agent is the
query engine**. See `SKILL.md` § Architecture.

## Provenance

Source: comisiondelaverdad.co — CC BY-NC-SA 4.0. This skill encodes **collective**
(ethical-political) responsibility as the CEV framed it, not individual criminal
guilt (the JEP's domain). The source PDFs are not bundled.
